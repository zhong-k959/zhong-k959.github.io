前言

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

准备工作

  • 安装git
  • 安装nodejs

安装情况

打开CMD命令下查看

环境配置

说明：这里的环境配置主要配置的是npm安装的全局模块所在的路径，以及缓存cache的路径，之所以要配置，是因为以后在执行类似：npm install express [-g] (后面的可选参数-g，g代表global全局安装的意思)的安装命令时，会将安装的模块安装到【C:\Users\用户名\AppData\Roaming\npm】路径中，占用C盘。——看个人需要

  • 在nodejs安装目录下创建 node_cache 和 node_global 两个文件夹
